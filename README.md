# Я Бао Завари — концепт сайта

Статический сайт для загрузки в GitHub/GitHub Pages.

## Структура
- `index.html` — разметка
- `css/styles.css` — стили и адаптив
- `js/app.js` — интерактив: 3D tilt, пар, фильтры меню, подбор настроения, галерея
- `assets/images` — оптимизированные WEBP-фото
- `assets/icons/logo.svg` — векторный знак в стиле логотипа

## Как загрузить на GitHub Pages
1. Создайте репозиторий.
2. Загрузите содержимое архива в корень репозитория.
3. Settings → Pages → Deploy from branch → main / root.
4. Откройте ссылку GitHub Pages.

Форма заявки демо-режимная. Её можно подключить к Telegram-боту, WhatsApp, Formspree, Bitrix24 или своему backend.
