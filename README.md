# Я Бао Завари — v5

Статический сайт для чайной: `index.html`, `catalog.html`, `privacy.html`.

## Что добавлено в v5

- телефон в шапке: на десктопе полный номер, на мобильной версии — компактная иконка;
- иконка корзины в шапке на главной и в каталоге;
- улучшенный UI/UX: быстрые действия, понятные CTA, аккуратные отступы, компактная мобильная шапка;
- блок основателя: Евгений Коркин + фото;
- более живой human-to-human текст без технических формулировок;
- реалистичнее SVG/3D-объекты: чашка с паром, чабань, глиняный чайник;
- нестандартная галерея с плотным grid;
- адаптивная проверка под 1920 / 1200 / 768 / 480;
- fallback для браузеров без backdrop-filter.

## Загрузка на GitHub Pages

Загрузите все файлы из архива в репозиторий и включите GitHub Pages для ветки `main` / root.

## Важно перед публикацией

Проверьте актуальность телефона, режима работы, ссылок на мессенджеры и Яндекс/2ГИС.
