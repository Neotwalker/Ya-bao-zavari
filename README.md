# Я Бао Завари — креативный сайт чайной

Статический сайт для GitHub Pages: HTML, CSS, JS.

## Что внутри

- адаптив под 1920 / 1200 / 768 / 480;
- 3D-элементы, параллакс-карточки и эффект пара;
- фильтр меню по настроению;
- мини-подбор формата визита;
- галерея с лайтбоксом;
- форма, которая формирует текст заявки и копирует его в буфер;
- страница политики конфиденциальности;
- подвал с годами работы и ссылкой на 2ГИС.

## Как опубликовать

Загрузите файлы в репозиторий GitHub и включите GitHub Pages для ветки `main`.

## Что заменить перед реальным запуском

1. Подставить реальные Telegram / WhatsApp / телефон, если они нужны в кнопках.
2. При необходимости подключить форму к Telegram Bot API, Formspree, Netlify Forms или CRM.
3. Добавить фото входа, если оно появится.


## Обновление v4

Добавлено:
- телефон в шапке с адаптивным скрытием на малых экранах;
- модальные окна для заявки на чайную церемонию и бронирования столика;
- 3D-стилизованные SVG-объекты: чашка с паром, чабань и чайник;
- кнопка Яндекс Карт рядом с 2ГИС;
- встроенная карта Яндекса;
- отдельная страница интернет-каталога `catalog.html`;
- корзина на localStorage и форма доставки с генерацией текста заказа;
- креативная нестандартная сетка галереи.

Форма и корзина сейчас формируют текст заявки. Для реальной отправки можно подключить Telegram Bot API, CRM, почту или сервис форм.
